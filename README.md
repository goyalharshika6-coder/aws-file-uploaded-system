# AWS File Upload System

A serverless file upload application built using AWS services.

## Technologies Used

- Amazon S3
- AWS Lambda
- API Gateway
- IAM
- HTML
- JavaScript

## Features

- Upload files directly to Amazon S3
- Secure uploads using Pre-Signed URLs
- API Gateway and Lambda integration
- CORS enabled
- Serverless architecture

## Architecture

User → API Gateway → Lambda → S3 Bucket

## Project Demo

- File selected from browser
- Lambda generates upload URL
- File uploaded directly to S3
- Stored securely in bucket
